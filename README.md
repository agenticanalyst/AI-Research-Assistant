# AI-Research-Assistant
An AI-powered Research Assistant built with n8n and Groq LLM. The agent uses conversational memory, performs calculations, retrieves information from Wikipedia, and can fetch data from external APIs using HTTP requests.
Features
```
🤖 AI Chat Assistant
🧠 Context-Aware Memory
🧮 Calculator Tool
📚 Wikipedia Integration
🌐 HTTP Request Support
⚡ Workflow Automation with n8n
🚀 Powered by Groq LLM
```
